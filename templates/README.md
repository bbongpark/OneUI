# templates/ — 산출물 템플릿

## 보고 PPT 템플릿 (회사에서 등록)

회사 표준 보고 양식 pptx를 이 폴더에 넣고, 텍스트 상자·표 안에 `{{자리표시자}}`를 심는다.
서버가 데이터를 채운 `report_fill.json`을 만들고 `scripts/ppt_fill.ps1`이 치환해 `data/<버전>/output/`에 저장한다.

### 지원 자리표시자 (서버 report_ppt 작업이 제공)

| 자리표시자 | 값 |
|---|---|
| `{{버전}}` | One UI 버전 |
| `{{전체건수}}` `{{P0건수}}` `{{P1건수}}` `{{P2건수}}` | 판정 분포 (rejected 제외) |
| `{{리뷰진행률}}` `{{PL통과율}}` `{{결정완료율}}` | 퍼센트 문자열 |
| `{{리스크건수}}` | 일정 리스크 high |
| `{{총평}}` | AI 작성 서술 (사람 확인 후) |
| `{{생성일}}` | 생성 시각 |

자리표시자를 추가하려면 `server/jobs.py`의 `job_report_ppt`에서 값을 추가하면 된다.
템플릿 파일이 늘어나면 대시보드 산출물 버튼도 자동으로 늘어난다 (파일명 = 버튼명).

## 변경점_템플릿.md

취합 단계에서 각 부서에 배포하는 변경점 열 작성 가이드. PL 검사가 준수 여부를 판정한다.
