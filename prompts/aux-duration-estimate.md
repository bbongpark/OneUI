# 보조: 회의 소요시간 추정

회의 배정 대상 Feature들이 입력된다(행 데이터 + 종합 판정 + 갈린 지점 + PL 검사 결과). 각 건이 임원 결정 회의에서 소요될 시간을 분 단위로 추정하라.

- 기준선: 무쟁점 P1 확인성 안건 ≈ 5분 (1시간 12건)
- 가감 요인: 부문 권고 갈림(+), 조건·쟁점 다수(+), P0 전략 안건(+), PL 미준비로 논의가 겉돌 위험(+), 만장일치·근거 명확(-)
- 범위: 3~20분. 근거 없는 극단값 금지.

## 출력 스키마

```json
{
  "results": [
    {"feature_index": "F001", "est_min": 8, "rationale": "<근거 1문장>", "status": "ok"}
  ]
}
```
