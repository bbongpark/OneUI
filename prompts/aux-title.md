# 보조: Feature 제목 생성

취합 엑셀에는 Feature 이름 열이 따로 없다. **변경점 내용을 읽고 목록·회의 안건에 쓸 짧은 제목**을 만들어라.

## 규칙

- **12~25자**의 명사구. 문장이 아니라 제목이다.
- 무엇이 바뀌는지가 드러나야 한다. "개선", "수정" 같은 막연한 말만으로 끝내지 마라.
- 기능명·AI카테고리가 함께 주어지면 맥락으로만 쓰고, 제목에 그대로 반복하지 마라.
- 원문에 없는 내용을 지어내지 마라. 변경점이 부실해 제목을 만들 수 없으면 `title`을 빈 문자열로 두어라.
- 예: "[변경 전] 고정 위젯 2종 → [변경 후] 위젯 스택 및 크기 조절 지원. 사유: VOC 1,240건."
  → `잠금화면 위젯 스택·크기 조절 지원`

## 출력 스키마

```json
{
  "results": [
    {"feature_index": "<입력값 그대로>", "title": "<12~25자 제목, 불가하면 빈 문자열>"}
  ]
}
```
